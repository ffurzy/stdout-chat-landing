Privacy Policy

// Last updated: March 7, 2026

stdout.chat is an anonymous chat application. This Privacy Policy explains what data we collect, how we use it, and what we do not do with it.

By using stdout.chat, you agree to the practices described in this policy.

// Age Requirement

stdout.chat is intended only for users 18 years of age or older.

By using this service, you confirm that you are at least 18 years old. We do not knowingly allow access to users under 18. If we become aware that a user under 18 has accessed the service, we may restrict access.

// What We Collect

stdout.chat is designed to collect as little data as possible.

// IP Address

Your IP address is processed for:

  • rate limiting
  • abuse prevention
  • automated moderation

IP addresses are not permanently stored, except when necessary to enforce bans.

// Device ID

A randomly generated anonymous Device ID is stored locally on your device (in the system Keychain). This identifier:

  • is randomly generated
  • is not derived from device fingerprinting
  • is not linked to your real identity

The Device ID is used only for:

  • the reputation / rating system
  • block lists
  • abuse prevention

// Rating Data

We store limited anonymous metrics associated with your Device ID:

  • rating score
  • number of conversations
  • average session duration

This information helps maintain conversation quality and prevent abuse.

// Reports

If you use the /report command:

  • we log the IP address of the reported user
  • we increment a report counter used by the automated moderation system

No message content is stored.

// Connection Metadata

We store minimal connection metadata for technical diagnostics:

  • connection timestamps
  • disconnection timestamps

This information is used to monitor server stability and detect abuse patterns.

// What We Do NOT Collect

stdout.chat intentionally avoids collecting personal data. We do not collect or store:

  • message content
  • conversation transcripts
  • user accounts
  • names, email addresses, or phone numbers
  • profile information

Conversations are ephemeral and exist only during the active chat session.

// Automated Moderation

stdout.chat uses an automated moderation system. If a user receives multiple reports:

  • automated restrictions may be applied
  • bans may be issued automatically based on report thresholds

This system operates without human review of message content because messages are never stored.

// Rating System

Your reputation rating is calculated based on conversation duration and reports received. The rating is stored anonymously on our servers and linked only to your Device ID.

You may view your rating at any time using the /score command. Your rating is never shared with third parties.

// Block List

When you use the /block command, the blocked user's Device ID is stored locally on your device only. stdout.chat does not store your block list on its servers.

// Cookies

stdout.chat does not use cookies for tracking. The website does not use tracking, advertising, or analytics cookies.

// Data Retention

  • Connection logs — retained for up to 7 days, then automatically deleted.
  • Rating data — retained while the Device ID remains active.
  • Ban records — retained for the duration of the ban, then deleted.
  • Report counters — retained only as long as necessary for moderation.

// Legal Basis for Processing

Where applicable under privacy laws such as GDPR, we process limited technical data under our legitimate interest in:

  • operating the service
  • preventing abuse
  • maintaining system security
  • ensuring service stability

// Security

We implement reasonable technical and organizational measures to protect the limited data processed by stdout.chat. However, no online service can guarantee absolute security.

// International Data Processing

stdout.chat may process limited technical data on servers located in the European Union or other jurisdictions depending on hosting infrastructure. By using the service, you consent to such processing.

// Third-Party Infrastructure

  • Vercel — hosts the website and may process standard web access logs.
  • Hetzner — hosts the chat server infrastructure.

These providers may process limited technical information necessary for infrastructure operation.

// Law Enforcement Requests

We may disclose limited technical data if required to do so by law or in response to valid legal requests from law enforcement authorities. Because stdout.chat does not store message content, we generally cannot provide conversation data.

// Your Rights

Depending on your jurisdiction, you may have certain privacy rights, including:

  • the right to request access to data associated with your IP address or Device ID
  • the right to request deletion of data where technically possible
  • the right to lodge a complaint with a data protection authority

Requests can be sent to hello@stdout.chat. Because stdout.chat does not store identifiable personal data, fulfilling some requests may not always be possible.

// Changes to This Policy

We may update this Privacy Policy from time to time. If changes are made, the updated version will be posted on this page with a revised Last updated date. Continued use of the service indicates acceptance of the updated policy.
